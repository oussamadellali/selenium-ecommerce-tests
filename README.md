# QA Automation - Selenium E-commerce Testing

## 🚀 Overview
Automated end-to-end testing project using Selenium WebDriver.

## 🧪 Test Scenarios
- Login (valid & invalid)
- Product selection
- Add to cart
- Checkout flow

## 🛠 Tech Stack
- Selenium WebDriver
- JavaScript (Node.js)
- Mocha & Chai
- GitHub Actions (CI/CD)

## ▶️ Run Tests
```bash
npm install
npm test
